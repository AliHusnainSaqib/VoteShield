# VoteShield

AI-Powered Digital Voting & Fair Electoral District System.

## Architecture
- **Frontend**: Next.js, Tailwind CSS, Framer Motion, React-Leaflet
- **Backend**: Python FastAPI
- **Database**: PostgreSQL with PostGIS

## Running the Application
1. Start the database and backend: `docker-compose up -d`
2. Start the frontend: `cd frontend && npm run dev`
